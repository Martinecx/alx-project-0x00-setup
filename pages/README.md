# ALX Project 0x00 - Next.js Airbnb Clone

This project is a Next.js-based application that mimics Airbnb's UI features.

## Features
- Property listings with images
- Ratings system
- Responsive design with Tailwind CSS
- TypeScript for type safety

## Getting Started
1. Clone this repository
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:3000

## Technologies Used
- Next.js
- TypeScript
- Tailwind CSS
- ESLint
